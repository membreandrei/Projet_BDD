  • Notes:
    • Le système de base de données est du MariaDB
    • Le langage utilisé pour communiquer avec le SGBD est du mysql, via le plugin jdbc

    • Informations sur les boutons du menu:
      • Afficher tous les programmeurs:
        • Un tableau contenant tous les programmeurs apparaît
        • Un double clic sur une des lignes du tableau donne les informations détaillées du programmeur
      • Afficher un programmeur:
        • Un tableau contenant tous les programmeurs apparaît
        • Un double clic sur une des lignes du tableau donne les informations détaillées du programmeur
        • La page contient également une fonction de recherche par ID, Nom, Prénom, ou année de naissance
      • Supprimer un programmeur:
        • Un tableau contenant tous les programmeurs apparaît
        • Un double clic sur une des lignes du tableau donne les informations détaillées du programmeur
        • La page contient également une fonction de recherche par ID
        • Un bouton de suppression est présent en bas de la page permettant de supprimer le(s) programmeur(s) sélectionné(s) dans le tableau
        • Il est possible de sélectionner des lignes du tableau avec la souris en maintenant le clic enfoncé
      • Ajouter un programmeur:
        • Ouvre une fenêtre contenant des champs décrivants les informations nécessaires à la génèse d'un programmeur
      • Modifier le salaire:
        • Un tableau contenant tous les programmeurs apparaît
        • La page contient également une fonction de recherche par ID
        • Un double clic sur une des lignes du tableau donne les informations détaillées du programmeur
        • Les champs de la fenêtre ouverte lors du double clic sont éditables et permettent de modifiers les caractéristiques du programmeur en question
      • Tous les menus:
        • Regroupe toutes les fonctionnalités de tous les autres menus de l'application

    • StyleHelper:
      • Classe dédiée à la mise en forme et au style de l'application
      • Crée dans un but de factorisation du code
      • Non finalisée en raison d'un léger manque de temps
