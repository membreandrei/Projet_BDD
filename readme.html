  • Notes:
    • Le système de base de données est du MariaDB
    • Le langage utilisé pour communiquer avec le SGBD est du mysql, via le plugin jdbc

    • Informations sur les boutons du menu:
      • Afficher tous les médias:
        • Un tableau contenant tous les médias
        • on peut suprimer un ou plusieurs média, et en ajouter un avec 2 boutons en bas
      • Média avec temps de parole des hommes 2x fois supérieur à celui des femmes:
        • Un tableau contenant tous les médias dont les temps de parole des hommes est 2x fois supérieur à celui des femmes
      • TV avec pourcentage temps de parole homme supérieur à X:
        • Un tableau contenant toutes les TV qui ont un pourcentage de temps de parole homme supérieur à X (la valeur de x par défaut est 0)
        • on peut rentrer le nombre dans le champ il verifie que ce que l'on tape est un nombre, et qu'il est compris entre 0 et 100
      • Pourcentage temps de parole:
        • Un tableau contenant le pourcentage des temps de parole par année
        • on peut rechercher l'année souhaité, elle verifie si un nombre entier est rentré
      • Moyenne temps de parole par média et par année:
        • Un tableau contenant tous les médias avec leur temps de parole par année
        • La page contient également une fonction de recherche par média il faut simplement entrer le nom du média
      • Importer à partir d'un csv:
        • permet de choisir un fichier .csv pour faire l'importation des données dans la base de donnée

    • StyleHelper:
      • Classe dédiée à la mise en forme et au style de l'application
      • Crée dans un but de factorisation du code
      • Non finalisée en raison d'un léger manque de temps
